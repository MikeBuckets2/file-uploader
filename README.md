# File Uploader

A personal file storage app. Upload files, organize them into folders, and share folders with an expiring link.

## Features

- Sign up and log in
- Create, rename, and delete folders
- Upload files to folders or to your root
- View file details (name, size, upload date)
- Download files
- Share folders with an expiring public link

## Tech Stack

- Node.js / Express
- EJS
- Prisma ORM
- PostgreSQL
- Passport.js

## File Limits

- Max file size: 10MB
- Allowed types: jpg, jpeg, png, gif, pdf, txt, doc, docx, xlsx, csv
